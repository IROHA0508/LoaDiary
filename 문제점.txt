문제점
1. 메인 페이지에서 레이드 생성 버튼을 눌렀는데 주소는 /raids/new가 뒤에 추가되는데 창이 바뀌지 않음
2. 유저 A가 유저 B의 캐릭터를 레이드에 추가했을 때, 유저 A의 브라우저에서는 레이드 파티가 보이지만 B의 브라우저에서는 새로고침을 해야 보임
새로고침을 하지 않아도 자동으로 감지해서 바뀌면 좋겠음 (실시간 연동이 될 것)
3. 그룹에 유저 A, B, C가 있을 때, 레이드 파티원 구성시 내 원정대 옆에 다음과 같은 순서대로 미리 원정대와 캐릭터가 보여서 사용자의 UX를 개선
배치 순서: 내 원정대 -> 그룹의 배치 순서대로 배치 -> 같은 그룹에 배치된 인원은 그룹 내부의 배치 순서대로 -> 최근에 이용한 원정대
기본적으로 위와 같은 형태로 로드 되어야 하며, 새로 추가할 원정대의 경우 내 원정대 오른쪽에 위치해야 함
4. 레이드 생성 후, 메인 페이지에서 레이드를 다시 클릭할 때 창이 바뀌지 않음
5. 4번의 경우에 이어서, 레이드 상세 페이지를 들어간 경우 내 캐릭터를 제외한 타인의 캐릭터가 배치되어 있지 않은것처럼 보임.
슬롯에 이름과 레벨은 없는데 색은 주황색으로 변경되는 사태가 있음
6. 레이드 상세 페이지에서 삭제를 눌렀는데 메인페이지에는 삭제되어야 하는 레이드가 그대로 존재함. 하지만 메인페이지에서 레이드 삭제 버튼을 눌렀을 때는 정상적으로 삭제됨
